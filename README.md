# Картель Долги бот для напоминаний о долгах

---

## Сам бот

Цель бота сделать большую сЕть серверов и начать крутой масштабный план по захвату дискорда картелем.  
Это часть исходного кода, и она **единственная**. Больше сливов не будет.

---

## Что внутри

В репозитории находится архив с исходным кодом бота и связанными ассетами.  
Код предназначен для развёртывания и дальнейшей дораБотКи.

---

## Как распаковать и использовать

1. Скачай архив из репозитория (`.zip`)
2. Распакуй архив в любую папку  
   - ПКМ → **Извлечь всё** (Windows)  
   - или любым архиватором
3. Открой папку с распакованными файлами
4. Убедись, что структура проекта сохранена (файлы не потерялись и не перемешались)
В основном там ненужные ассеты еще
