## Что внутри

В репозитории находится файлы исходный сурс кодик рандомных сообщений
и их функции для отправки (типо кусочек от исходного кода картель долги)

---

## Как распаковать и использовать

1. Скачай архив из репозитория (`.zip`)
2. Распакуй архив в любую папку  
   - ПКМ → **Извлечь всё** (Windows)  
   - или любым архиватором
3. Открой папку с распакованными файлами
А в основном там ненужные ассеты лежат
